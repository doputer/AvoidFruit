# 2D 과일피하기 게임
> 유니티로 제작한 2D 과일피하기 게임 소스입니다.

## 게임 방법

좌우 방향키(이동)와 스페이스 바(점프)를 이용해서 하늘에서 떨어지는 과일을 피하는 게임입니다. 과일을 많이 피하면 점수가 높게 상승하고, 과일에 세 번 부딪히면 게임에서 패배하게 됩니다.
